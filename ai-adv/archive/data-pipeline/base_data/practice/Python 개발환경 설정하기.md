# [실습] 파이썬 개발환경 설정하기
파이썬과 VS Code를 설치했다면, 이제 파이썬 개발 환경의 사용법을 알아보겠습니다!   
각 프로그램은 아래의 경로에서 다운로드 및 설치 가능합니다.

- https://www.python.org/downloads/
  - 설치 시 'Add Python to PATH'를 꼭 체크해 주세요.
- https://code.visualstudio.com/ 

## 1. 파이썬 실행하기   

OS의 터미널(또는 명령 프롬프트)를 실행하거나, VS Code에서 Ctrl+J를 입력하여 하단 터미널을 열어주세요.  

만약 VS Code를 파이썬 설치보다 먼저 실행한 경우, 프로그램을 한번 종료했다가 켜 주세요.

1. `python --version` 커맨드 실행   

정상적으로 출력이 되면 준비가 된 것입니다.   
`python` 커맨드를 실행해 터미널에서 파이썬 창을 열어주세요.
2. 샘플 Python 코드(.py) 만들기

오른쪽 Explorer 탭을 열고, hello.py 파일을 만들어 주세요.   
`print('hello world!')`를 입력합니다.
3. 코드 실행하기   
코드 창에서 Shift + Enter를 입력하면 코드가 실행됩니다.
## 2. ipynb 파일의 이해

파이썬의 기본 파일 형태는 py 파일이지만,   
본 과정에서는 주로 주피터 노트북 (.ipynb) 파일을 사용합니다.   

ipynb 파일은 전체 코드가 아닌 셀(Cell) 단위로 코드를 실행하며,   
커널(Kernel)에 해당 코드의 실행 내용이 저장되는 방식입니다.   

아래 코드를 실행해 보겠습니다.   
셀을 클릭한 뒤, Ctrl+ Enter 또는 Shift + Enter를 실행해 주세요.    
(왼쪽의 화살표를 클릭해도 됩니다.)
print('Hello Jupyter Notebook!')
실행하면 상단에 커널 선택 창이 나타납니다.   
Python Environments를 클릭합니다.   
여기서 가상 환경을 사용할지, 방금 설치한 Python을 그대로 사용할지 선택할 수 있습니다.

- PC에 설치된 Python 사용: 한 번 설정해두면 여러 프로젝트에서 공통으로 활용할 수 있습니다.

- 가상 환경 생성: 현재 프로젝트 전용의 독립적인 Python 환경을 구성합니다.    
라이브러리 버전 충돌을 막고, 프로젝트별 버전 관리를 쉽게 할 수 있다는 장점이 있습니다.   

선택한 커널은 나중에 오른쪽 버튼이나 F1 --> Select Python Interpreter를 통해 변경할 수 있습니다.
