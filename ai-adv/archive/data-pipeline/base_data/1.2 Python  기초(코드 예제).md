## 2.변수·자료형·조건·반복 샘플

### 변수·형변환

```python
x = "123"
n = int(x) # "123" → 123
s = str(100) # 100 → "100"
```
- 포인트: `int()`, `str()` 형변환, 동적 타이핑.

### Truthy / Falsy 체크

```python
value = []
if value:
    print("Truthy")
else:
    print("Falsy")
```
- 포인트: `[]`,`""`,`0`,`None` 등은 Falsy.

### 조건문

```python
score = 85

if score >=90:
    grade = "A"
elif score >= 80:
    grade = "B"
else:
    grade = "C"    
```
-포인트: `if / elif / else` 구조, 조건식 True/False.

### 논리연산자

```python
age = 25
is_student = True

if age >=20 and is_student:
    print("성인 학생")
```
- 포인트: `and`, `or`, `not` 의미.

### for + range / enumerate / zip

```python
for i in range(5)
    print(i)
```

```python
items = ["a", "b", "c"]
for idx, value in enumerate(items):
    print(idx, value)
```

```python
names = ["철수", "영희"]
scores = [90, 95]
for name, score in zip(names, scores):
    print(name, score)
```
- 포인트: `range()`, `enumerate()`, `zip()` 사용법과 반환값 형태.

### while + break/continue/else

```python
n = 0
while n < 5:
    n += 1
    if n == 3:
        continue
    if n == 4:
        break
    print(n)
else:
    print("정상 종료")
```
- 포인트: `continue`는 다음 반복, `break`는 종료, `while-else`는 break 없이 끝날 때 실행.

## 3. 함수·모듈 샘플

### 중복 제거 함수(DRY)

```python
def normalize_text(text):
    return text.strip().lower().replace(" ","")

user_input1 = normalize_text(" Hello ")
user_input2 = normalize_text("WORLD ")
```
- 포인트: `strip()`, `lower()`, `replace()` 체이닝, 함수로 로직 추상화.

### 기본적인 함수 구조

```python
def add(a,b):
   """두 수를 더해서 반환한다."""
   result = a + b
   return result
```
- 포인트: 시그니처, docstring, 반환값.

### 매개변수 종류

```python
def greet(name, age):
    return f"{name}님음 {age}살입니다"

greet("김철수", 25)          # 취치 인지
greet(age=25, name="김철수") # 키워드 인지
```

```python
def connect(host="localhost", port=3306):
    print(host, port)
```

```python
def log(*args, **kwargs):
    print(args)     # 위치 인자 튜플
    print(kwargs)   # 키워드 인자 딕셔너리
```
- 포인트: 위치 인자, 키워드 인자, 기본값, `*args`, `**kwargs` 의미.

### 모듈·import

```python
# math_utils.py
def add(a, b):
    return a + b
```

```python
# main.py
import math_utils
print(math_utils.add(2, 3)) # 5
```
- 포인트: 파일 = 모듈, `import 모듈명·함수` 호출.

### 패키지 import

```python
# 폴더 구조
my_package/
    __init__.py
    math_utils.py
    string_utils.py
```

```python

```
- 포인트: 패키지 구조, `from 패키지 import 모듈` 문법.

## 4. 리스트 컴프리헨션·파일·JSON·예외 샘플

### 리스트 컴프리헨션

```python
result = []
for x in range(10):
    if x % 2 == 0:
        rerult.append(x**2)
```

```python
result = [x**2 for x in range(10) if x % 2 == 0]
```
- 포인트: 동일 로직을 한 줄로, 순서: 식 for 변수 in 반복 if 조건.

### 파일 입출력(with)

```python
# 읽기
with open("example.txt", "r", encoding="utf-8") as f:
    text = f.read()

# 쓰기
with open("example.txt", "w", excoding="utf-8") as f:
    f.wirte("Hello, File!")
```
- 포인트: `open()` 모드 `"r"`, `"w"`, `"a"`, `"b"`: `with` 로 자동 close.

### JSON 처리

```python
import json

data = {"name": "철수", "age":25}
json_str = json.dumps(data.ensure_ascii=False, indent=2)

data2 = jaon.loads(json_str)
```
- 포인트: `dumps`(dict→문자열), `loads`(문자열→dict), `ensure_ascii`, `indent` 옵션.

### 예외·try-except

```python
today = 100
yesterday = 0
change_rate = (today - yesterday) / yesterday * 100 # ZeroDivisionError
```

```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print("0으로 나눌 수 없습니다.")
except Exception as e:
    print("예상치 못한 오류:", e)
else:
    print("에러 없이 실행됨")
finally:
    print("항상 실행되는 코드")
```
- 포인트: 구체 예외 → 일반 `Exception` 순서, `else`, `finally` 역할.

## 5. 클래스·상속 샘플

### 기본 클래스·생성자·메서드

```python
class Person:
    def __init__(self, name):
        self.name = name

    def greet(self):
        return f"안녕하세요. 저는 {self.name}입니다."
p = Person("철수")
print(p.greet())
```
-포인트: `__init__`,`self`, 인스턴스 생성, 매서드 호출.

### 클래스 변수 vs 인스턴스 변수

```python
class Counter:
    total = 0 # 클래스 변수

    def __init__(self):
        self.count = 0 # 인스턴스 변수
        Counter.total += 1
```
- 포인트: `Counter.total`처럼 클래스명으로 접근, `self.count`는 인스턴스별.

### 상속·메서드 추가

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def eat(self):
        return f"{self.name}가 먹고 있습니다."

class Dog(Animal):
    def bark(self):
        return f"{self.name}: 멍멍!"

class Cat(Animal):
    def meow(self):
        return f"{self.name}: 야옹~"
```
- 포인트: `class Dog(Animal)` 문법, 부모 메서드(`eat`) 재사용 가능.

### 오버라이딩·super()

```python
class Parent:
    def greet(self):
        return "부모 인사"

class Child(Parent):
    def greet(self):
        return super().greet() + " + 자식 인사"
```
- 포인트: 같은 이름 메서드 재정의 = 오버라이딩, `super().greet()`로 부모 기능 + 확장.
