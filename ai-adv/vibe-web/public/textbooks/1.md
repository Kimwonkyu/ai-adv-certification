## 2. 변수와 자료형 (Variables & Data Types)
### 📦 기본 자료형
데이터를 담는 그릇인 변수는 다양한 형태를 가집니다.
- **수치형**: `int`(정수), `float`(실수).
- **문자열**: `str`. 큰따옴표("")나 작은따옴표('')로 감쌉니다.
- **불리언**: `bool`. `True` 또는 `False`. (첫 글자 대문자 주의)

### 📚 컨테이너 자료형 (Collection)
여러 개의 데이터를 묶어서 관리합니다.
1.  **List (`[]`)**: 수정 가능(Mutable). 순서가 있습니다.
    - `my_list = [1, 2, 3]`
    - `my_list.append(4)`: 요소 추가
2.  **Tuple (`()`)**: **수정 불가능(Immutable)**. 순서가 있습니다.
    - `my_tuple = (1, 2)`
    - 읽기 전용 데이터나 딕셔너리의 키로 사용할 때 유용합니다.
3.  **Dictionary (`{}`)**: 키(Key)와 값(Value)의 쌍으로 저장합니다.
    - `my_dict = {"name": "Kim", "age": 25}`
    - `my_dict["name"]`으로 빠르게 값을 찾을 수 있습니다. (`O(1)` 속도)

## 3. 제어문 (Control Flow)
프로그램의 흐름을 제어합니다.
### 🔀 조건문 (if)
```python
score = 85
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("C")
```

### 🔁 반복문 (for, while)
- **for**: 정해진 횟수나 리스트의 요소만큼 반복합니다.
    ```python
    for i in range(5):  # 0부터 4까지
        print(i)
    ```
- **List Comprehension (심화 보충)**: 리스트를 간결하게 만드는 파이썬만의 강력한 문법입니다.
    ```python
    # 일반적인 방식
    squares = []
    for x in range(10):
        squares.append(x**2)
    
    # List Comprehension
    squares = [x**2 for x in range(10)]
    ```

## 4. 함수와 모듈 (Functions & Modules)
### 🔨 함수 (Function)
반복되는 코드를 묶어서 재사용성을 높입니다.
```python
def greet(name="Guest"):  # 기본값 설정 가능
    return f"Hello, {name}!"

print(greet("Alice"))
```

### 📦 모듈과 패키지
- **모듈(Module)**: `.py` 파일 하나. 다른 파일에서 `import`하여 함수나 변수를 가져다 쓸 수 있습니다.
- **패키지(Package)**: 여러 모듈을 모아둔 폴더.
- **라이브러리(Library)**: 패키지들의 묶음. (예: `numpy`, `pandas`)

## 5. 객체지향 프로그래밍 (OOP) 기초
파이썬의 모든 것은 객체(Object)입니다.
### 🏗 클래스(Class)와 인스턴스(Instance)
- **클래스**: 붕어빵 틀(설계도). 속성(변수)과 행위(함수)를 정의합니다.
- **인스턴스**: 붕어빵(실체). 클래스를 통해 만들어진 실제 객체입니다.

```python
class Dog:
    def __init__(self, name):  # 생성자: 객체 생성 시 자동 실행
        self.name = name       # 속성(Attribute)

    def bark(self):            # 메서드(Method)
        return f"{self.name} says Wal!"

my_dog = Dog("Happy")
print(my_dog.bark())
```

### 🧬 상속 (Inheritance)
기존 클래스(부모)의 기능을 물려받아 새로운 클래스(자식)를 만듭니다. 중복 코드를 줄일 수 있습니다.

## 6. 예외 처리와 파일 입출력
### 🛡 예외 처리 (try-except)
프로그램 실행 중 에러가 발생해도 꺼지지 않게 막아줍니다.
```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print("0으로 나눌 수 없습니다.")
```

### 📂 파일 입출력 (File I/O)
`with` 문을 사용하면 파일을 열고 닫는 것(`close`)을 자동으로 처리해줍니다. (권장)
```python
# 파일 쓰기
with open("data.txt", "w", encoding="utf-8") as f:
    f.write("Hello Python")

# 파일 읽기
with open("data.txt", "r", encoding="utf-8") as f:
    content = f.read()
```
