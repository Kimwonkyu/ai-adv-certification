# 📘 [학습 노트] 교재 1. Python 기초 (Foundation to Advanced)

이 교재는 파이썬의 기초 문법부터 메타프로그래밍, 메모리 관리, 동시성 구조까지 모든 난이도의 문항(720문항)을 완벽히 대비할 수 있도록 구성되었습니다.

## 1. 파이썬 입문 및 환경 설정
### 🐍 파이썬의 특징과 철학
- **특징**: 간결한 문법, 동적 타이핑(Dynamic Typing), 풍부한 라이브러리('Batteries Included').
- **Duck Typing**: 객체의 타입보다 객체가 가진 메서드나 속성이 중요합니다.
- **EAFP**: "허락보다 용서가 쉽다." `try-except`를 적극 활용하는 파이썬의 철학입니다.
- **환경 설정**: PEP 8 스타일 가이드(snake_case, 4칸 들여쓰기), 가상환경(`venv`), `if __name__ == "__main__":` 등의 실행 구조를 숙지해야 합니다.

## 2. 자료형과 메모리 (Deep Dive)
### 📦 Mutable vs Immutable (가변 vs 불변)
파이썬 메모리 관리의 핵심 개념입니다.
- **Immutable(불변)**: `int`, `str`, `tuple`. 값을 수정하면 새로운 객체가 생성(새로운 `id`)됩니다.
- **Mutable(가변)**: `list`, `dict`, `set`. 기존 객체의 내용을 직접 수정할 수 있습니다.
- **id()**: 객체의 유일한 식별자(메모리 주소)를 반환합니다.

### 🧬 기본 자료구조의 특성
- **List**: Dynamic Array. 인덱스 접근 $O(1)$, 삽입/삭제 $O(n)$.
- **Dict**: Hash Table. 키 조회 평균 $O(1)$. 3.7+ 버전부터 입력 순서가 보장됩니다.
- **Set**: 중복 불허 Hash Table. 집합 연산에 최적화되어 있습니다.

## 3. 제어문, 함수, 그리고 스코프
### 🔄 제어 흐름
- **Walrus Operator (`:=`)**: 조건문 내에서 할당과 검사를 동시에 수행합니다.
- **Generator**: `yield`를 사용하여 메모리 효율을 극대화하며 데이터를 하나씩 생성합니다.

### 🌐 변수 범위 (LEGB 규칙)
변수를 찾는 순서: **L**ocal → **E**nclosing → **G**lobal → **B**uilt-in.
- `global`, `nonlocal` 키워드를 통해 상위 스코프의 변수를 수정할 수 있습니다.

---

## 4. 메타프로그래밍 (Metaprogramming)
### 🏗 Metaclass와 `type`
- **Metaclass**: 클래스 그 자체를 생성하는 팩토리입니다. 모든 클래스의 기본 메타클래스는 `type`입니다.
- **생성 루틴**: `__new__` (메모리 할당) → `__init__` (초기화).

### 📜 Descriptor Protocol
속성 접근(`obj.attr`)을 가로채는 메커니즘입니다.
- **Data Descriptor** (`__set__` 구현)가 **인스턴스 딕셔너리**(`__dict__`)보다 우선순위가 높습니다.

---

## 5. 상속과 MRO (Method Resolution Order)
- **C3 Linearization**: 다중 상속 시 메서드 호출 순서를 결정하는 알고리즘입니다.
- **`super()`**: 단순히 부모를 부르는 것이 아니라, MRO 상의 다음 클래스를 호출합니다.

---

## 6. 메모리 관리와 성능
### 🔒 GIL (Global Interpreter Lock)
- 한 번에 하나의 스레드만 바이트코드를 실행하게 제한하는 뮤텍스입니다. CPU Bound 작업 시 멀티프로세싱이 멀티스레딩보다 유리한 이유입니다.

### 📦 최적화와 GC
- **`__slots__`**: `__dict__` 생성을 막아 객체당 메모리 점유율을 줄입니다.
- **Generational GC**: 순환 참조를 해결하기 위한 세대별 가비지 컬렉션입니다.

---

## 7. 동시성 및 고급 기법
- **Asyncio**: 이벤트 루프 기반 비동기 I/O. `await`로 I/O 대기 중 제어권을 이양합니다.
- **Decorator**: `functools.wraps`로 메타데이터를 보존하며 기능을 확장합니다.
- **Context Manager**: `with` 문을 통해 자원의 안전한 획득과 해제를 보장합니다.
