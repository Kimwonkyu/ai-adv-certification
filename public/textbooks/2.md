## 1. NumPy: 수치 연산의 기본
파이썬 리스트보다 훨씬 빠르고 강력한 수치 계산 라이브러리입니다.

### 🔢 배열(Array) 생성과 기초
- **`np.array`**: 리스트를 배열로 변환합니다. 모든 요소는 같은 자료형이어야 합니다.
- **초기화 함수**: `np.zeros(10)`, `np.ones((2,3))`, `np.arange(0, 10, 2)` 등으로 쉽게 배열을 만듭니다.

### ➕ 벡터 연산 (Vectorization)
반복문(`for`) 없이 배열 전체에 연산을 적용할 수 있습니다.
```python
import numpy as np
arr = np.array([1, 2, 3])
print(arr * 10)  # [10, 20, 30] - 요소별 연산
```

## 2. Pandas: 데이터프레임 다루기
엑셀과 같은 테이블 형태의 데이터를 다루는 핵심 도구입니다.

### 📊 기본 구조
- **Series**: 1차원 데이터 (세로로 된 한 줄).
- **DataFrame**: 2차원 데이터 (행과 열이 있는 표). 여러 Series가 모인 것과 같습니다.

### 🔍 데이터 선택과 필터링
- **`df['컬럼명']`**: 특정 열 선택.
- **`df.loc[행이름, 열이름]`**: 이름(Label)을 기준으로 선택.
- **`df.iloc[행번호, 열번호]`**: 순서(Index)를 기준으로 선택.
- **조건 필터링**:
    ```python
    # 나이가 20세 이상인 사람만 추출
    adults = df[df['age'] >= 20]
    ```

---

## 3. 데이터 전처리 (Preprocessing)
### 🧹 결측치(Null) 처리
데이터가 비어있는 경우를 처리하는 것은 매우 중요합니다.
- **`dropna()`**: 결측치가 있는 행/열을 삭제합니다.
- **`fillna(값)`**: 결측치를 특정 값(0, 평균 등)으로 채웁니다.

### 🔄 데이터 변환
- **`apply()`**: 함수를 데이터프레임의 행이나 열 단위로 적용합니다.
    ```python
    df['name_len'] = df['name'].apply(len)  # 이름 길이 계산
    ```
- **`astype()`**: 데이터 타입을 변환합니다. (예: 문자열 "100" → 숫자 100)

---

## 4. 그룹화와 집계 (GroupBy)
데이터를 그룹별로 묶어서 통계를 냅니다. (엑셀의 피벗 테이블과 유사)
```python
# 성별(sex)로 묶어서 나이(age)의 평균(mean) 구하기
df.groupby('sex')['age'].mean()
```

---

## 5. 데이터 병합 (Merge)
두 개의 데이터프레임을 하나로 합칩니다. (SQL의 JOIN과 유사)
- **`pd.concat`**: 단순히 위/아래나 옆으로 이어 붙입니다.
- **`pd.merge`**: 공통된 열(Key)을 기준으로 데이터를 합칩니다. (`on`, `how` 옵션 중요)
