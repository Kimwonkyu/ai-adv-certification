이 교재는 NumPy와 Pandas의 기본 API 사용법부터 하이엔드 성능 최적화, 머신러닝 전처리의 수리적 무결성까지 모든 난이도의 문항을 커버합니다.

---

## 1. NumPy: 수치 계산의 기초와 심화
### 🛠 기본 속성 및 생성
- **기본 속성**: `shape` (차원 구조), `dtype` (데이터 타입), `ndim` (차원 수).
- **데이터 생성**: `np.array`, `np.zeros`, `np.ones`.
- **`arange()` vs `linspace()`**: `arange`는 간격을, `linspace`는 개수를 기준으로 생성하며 `linspace`가 부동 소수점 오차 방지에 유리합니다.

### ⚡ 고성능 연산 원리
- **SIMD (Single Instruction Multiple Data)**: CPU 하드웨어 단에서의 병렬 처리.
- **UFuncs**: C로 구현된 유니버설 함수를 통한 벡터화 연산.
- **메모리 레이아웃 (C vs F-style)**: 인접 데이터 접근 효율(Cache Locality) 결정.
- **`flatten()` vs `ravel()`**: 복사본 생성 여부에 따른 메모리 효율 차이.

---

## 2. Pandas: 구조화 데이터 처리
### 📂 데이터 입출력 및 탐색
- **IO**: `pd.read_csv`, `df.to_csv(index=False)` (무의미한 행 번호 제거).
- **탐색**: `df.head()`, `df.info()`, `df.describe()` (통계량 요약).
- **선택**: `df['col']` (Series), `df[['col']]` (DataFrame 형태 유지).

### 🛠 데이터 가공 및 정제
- **결측치**: `df.isnull().sum()`, `df.fillna()`, `df.dropna()`.
- **날짜 처리**: `pd.to_datetime(errors='coerce')`로 무효한 데이터 결측 처리.
- **인덱싱**: `reset_index(drop=True)`로 정수 인덱스 초기화 및 조회 성능 $O(1)$ 확보.
- **병합**: `pd.concat(join='outer'|'inner')` (합집합 vs 교집합).

---

## 3. 통계적 전처리와 피처 엔지니어링
### 📈 스케일링 (Scaling)
- **StandardScaler** (평균 0, 분산 1), **MinMaxScaler** (0~1 사이).
- **Data Leakage 주의**: `fit()`은 반드시 Train 세트에만 수행할 것.

### 📐 차원 축소 (PCA)
- **Mean Centering**: 주성분 분석 전 데이터를 원점으로 이동시켜 최대 분산 방향(PC1)을 정확히 포착하게 함.

### 🚩 범주형 데이터 처리
- **One-hot Encoding**: 카테고리를 이진 벡터로 변환.
- **Dummy Variable Trap**: 다중공선성 방지를 위해 $N-1$개의 열만 사용해야 함.
